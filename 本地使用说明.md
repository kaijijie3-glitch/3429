# 💻 本地使用和部署说明

## 🎯 第一步：本地运行（最简单）

### Windows 用户
1. 直接双击 `启动.bat` 文件
2. 等待启动完成
3. 在浏览器打开：http://localhost:5174

### 手动启动
```bash
# 启动后端（新窗口）
cd server
npm install
node index.js

# 启动前端（另一个新窗口）
cd client
npm install
npm run dev
```

## 🌐 第二步：部署到互联网

### 方案A：使用 Railway（最简单，免费）

1. **注册账号**
   - 访问：https://railway.app
   - 用 GitHub 账号登录

2. **推送到 GitHub**
   ```bash
   cd d:\123456789
   git init
   git add .
   git commit -m "first commit"
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   git push
   ```

3. **在 Railway 部署**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你的仓库
   - 等待自动部署完成

### 方案B：使用 Vercel（推荐，简单）

前端部署到 Vercel，后端部署到 Railway：

1. **前端部署到 Vercel**
   - 访问：https://vercel.com
   - 导入你的 GitHub 项目
   - 配置环境变量（API地址）
   - 一键部署

2. **后端部署到 Railway**
   - 同上

## ❓ 常见问题

### Q: IDE里的"部署"按钮用不了？
A: Trae IDE 是开发工具，不提供部署功能。部署需要用专业的部署平台。

### Q: 本地运行别人能访问吗？
A: 不能。本地运行只能在你自己的电脑访问。要让别人访问，需要部署到服务器。

### Q: 推荐什么部署平台？
A: **Railway** - 最简单，免费额度足够使用。

### Q: 需要域名吗？
A: 不需要。Railway 会免费给你一个临时域名。
